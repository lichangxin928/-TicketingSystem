		系统各个模块简介

			用户管理模块：
				用户可以通过电子邮箱，进行票多多影院账号注册，输入合法的用户名，电子邮箱以及自己设置的密码，便可以进行账号注册。通过登录账号，便可以进入票多多影院售票系统进行购买电影票。用户也可以更改自己的密码，通过进入个人主页，点击修改密码，输入以前的密码，再输入新密码，确认无误后便可以成功修改密码。用户如果忘记密码，可以通过电子邮箱进行重新设置密码。通过电子邮箱发送的验证码，进行验证，便可以重新设置密码，用户用新密码便可以重新登录进入售票系统。

			管理员模块：
				管理员输入管理员账号和密码进入管理员页面，管理员页面能够进行对用户信息修改、删除用户、新增用户等等，还能够对购票记录进行统计，查看哪些用户买了哪些票。

			个人主页模块：
				个人主页可以在用户登陆成功以后通过按键进入个人空间，里面可以实现用户对账户密码的管理，进行密码修改等功能；还有查看用户的个人订单，实现对购票历史记录的查看，方便用户管理订单；个人主页中还有可以供用户浏览的电影精选图片。

			售票系统模块：
				用户可在登陆成功以后通过点击想要购买的电影进入该网页，然后点击购票即可实现购票，在订单和管理员历史记录都可以看见；还要电影专栏可浏览其他电影，选择自己想看的电影

		
		源代码文件简述：

			views文件夹：各个页面的 html 文件
			public文件夹：公共的文件


			views/user/user_login.html 用户登录页面
			views/user/menu.html 主界面菜单
			views/user/change_success.html 修改密码成功页面
			views/user/changePsw.html 修改密码界面
			views/user/forgot.html 忘记密码页面
			views/user/home.html  个人主页
			views/user/order.html   个人菜单
			.....

			public/ 公用的资源，涉及到图片文件，css 文件，js 文件，框架依赖等等

			package.json 在创建项目时自动生成的对项目引用的 node 的包的介绍以及大致框架
			package-lock.json 项目包的相互依赖介绍

			main.js 为后端程序的入口，通过 node 平台启动服务器，让前端平台能够访问
